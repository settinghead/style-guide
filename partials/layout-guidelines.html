Layout Guidelines

Basics of Layout

Container has an inner padding of 15px

Row has a negative horizontal margin of 15px. Therefore, it spills to the outer limit of the container and breaks the layout.

Column has a horizontal padding 15px.

This accounts for the negative margin of the row and therefore, resets the layout.

Containers such as div and form when not used with grid classes can be placed in the main container without breaking the layout.

Nested Grid

Ensure that all column classes, at each level, are wrapped in a row div.

Level 1: .col-sm-9
Level 2: .col-xs-8 .col-sm-6
Level 2: .col-xs-4 .col-sm-6
