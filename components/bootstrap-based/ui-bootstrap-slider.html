UI Bootstrap Slider

An Angular directive for making use of the bootstrap-slider component. The angular-bootstrap-slider directive supports different orientations (horizontal/vertical), custom formatter, range selector, and more. This directive can support more complex examples, for more understanding on the library being used by the directive, see bootstrap-slider. Note, this directive currently has a dependency on version 3.1.0 of the bootstrap-slider library

Dependencies Bootstrap bootstrap-slider Angular angular-bootstrap-slider

Module Includes

Declare a dependency on the ui-bootstrap in your module:

angular.module('myApp', ['ui.bootstrap-slider']);

Script Includes

Make sure you include the following files in index.html

bower_components/bootstrap/dist/js/bootstrap.js bower_components/angular/angular.js bower_components/seiyria-bootstrap-slider/dist/bootstrap-slider.min.js bower_components/angular-bootstrap-slider/slider.js
