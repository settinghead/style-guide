Modal

Standard for modal usages using ui-bootstrap, see UI-Bootstrap Modals for more info.

Dependencies AngularJS UI Bootstrap

Module Includes

Declare a dependency on the ui. in your module:

angular.module('myApp', ['ui.bootstrap']);

Script Includes

Make sure you include the following files in index.html

bower_components/angular/angular.js bower_components/angular-bootstrap/ui-bootstrap.js
