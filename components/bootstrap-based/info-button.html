Info Button

This is intended to be used in page/section headers to show info about a section or a group. For Risevision, the trigger should be set to 'click' so that it will work on mobile devices. This uses the UI Popover and the library can be used to build more complex examples, see UI Popover for more info.

Dependencies RV-Common-Style AngularJS UI Bootstrap

Module Includes

Declare a dependency on the ui. in your module:

angular.module('myApp', ['ui.bootstrap']);

Script Includes

Make sure you include the following files in index.html

bower_components/angular/angular.js bower_components/rv-common-style/rise.min.css bower_components/angular-bootstrap/ui-bootstrap-tpls.js
